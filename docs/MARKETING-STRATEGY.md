## 🎨 Content Strategy Framework

### Promotion Hierarchy (Tested & Validated)

**Tier 1: High Performers** (Do More)
- BOGO (Buy 1 Get 1) — Clear, tangible value
- Weekend specials — Timely, FOMO-inducing
- Food-focused visuals — Appetite appeal

**Tier 2: Moderate Performers** (Optimize)
- % discounts — Needs clear math (e.g., "50% OFF" not "20% OFF")
- Flash sales — Urgency works, but don't overuse
- Happy hour — Time-bound, location-specific

**Tier 3: Low Performers** (Avoid or Test Heavily)
- Complex promotions — Confusing, low conversion
- Generic posts — No clear CTA or offer
- Text-heavy images — Low visual appeal

### Headline Formulas (Proven Winners)

```
1. [EMOJI] [BENEFIT] + [URGENCY]
   Example: "🔥 BOGO ALL MAINS — Weekend Only!"

2. [PRICE] + [PRODUCT] + [SCARCITY]
   Example: "$10 Thai Fried Rice — Today Only!"

3. [FOMO] + [SOCIAL PROOF] + [CTA]
   Example: "Everyone's coming this weekend! Join the feast 🎉"
```

### Visual Best Practices

**DO:**
- Close-up food shots (steam, texture, color)
- Bold, readable text overlays
- Warm, appetizing colors (red, orange, gold)
- Single focal point (one hero dish)

**AVOID:**
- Stock photos (inauthentic)
- Cluttered compositions
- Hard-to-read text
- Dark or cold colors

## 📞 Decision Framework

### When to Pivot
- **3 consecutive underperforming posts** → Review strategy
- **New pattern emerges (3+ times)** → Update best practices
- **Analytics contradict hypothesis** → Trust data, adjust theory

### When to Double Down
- **Post performs 2x average** → Analyze why, replicate
- **Format wins 3+ times** → Make it a staple
- **Time slot consistently wins** → Prioritize it

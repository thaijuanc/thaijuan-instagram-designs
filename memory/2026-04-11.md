## ⚠️ LESSONS LEARNED (2026-04-11)

**Canva MCP Auth:**
- Tokens expire after 3.5 hours
- Auto-refresh cron runs every 2 hours (0 */2 * * *)
- MUST run commands from `/Users/fenton/.openclaw/workspace` (mcporter.json location)
- If auth fails: run `mcporter auth canva-mcp` once, then continue

**Mobile-First Formatting:**
- NO decorative dashes (───)
- NO extra blank lines
- Condensed, scannable content only
- Respect user's screen space

**Workflow Optimization (v3.0):**
- Generate 1 design instead of 4 (saves ~2-3 min)
- Skip Google Calendar event (saves ~30 sec)
- Attach PNG directly in Discord for preview (not just URL)
- Total time: ~4-5 min per post (down from ~7-8 min)

**Clear System Feedback (NEW):**
- BEFORE starting: Show full plan + Step 1 draft for approval
- BEFORE each step: State what you're about to do + estimated time
- THEN execute the step
- User is never left wondering "what's happening?" or "how long?"

**Canva MCP Prompt (NEW):**
- Show the exact optimized prompt before generating
- Include: subject, style, colours, mood, text overlay, composition
- User can approve or tweak before generation

**Instagram URL Fix (2026-04-11):**
- Issue: Notifications sent broken links with numeric post IDs (e.g., `instagram.com/p/18584627212016182`)
- Root cause: Instagram uses alphanumeric shortcodes, not numeric IDs, in URLs
- Fix: `instagram-poster.js` now calls `fetchInstagramPermalink()` after posting to get proper shortcode URL from Graph API
- Files updated: `instagram-poster.js`, `notify-juan.js`
- Result: All future notifications will have working Instagram links like `instagram.com/p/ABC123xyz/`
